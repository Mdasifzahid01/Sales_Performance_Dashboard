
# 📊 E-Commerce Insights Dashboard

## 🚀 Project Overview

An interactive Power BI dashboard designed to analyze e-commerce sales data and generate meaningful business insights.

## 🎯 Objective

To identify top-performing regions, categories, and sales trends to support better business decisions.

## 🛠️ Tools Used
* Power BI
* Excel
* Pandas
* SQL
* 
## 📈 Dashboard Features
* KPI Cards (Sales, Profit, Quantity)
* Region, Category, and Year Filters
* Sales by Category & Region
* Monthly Sales Trend
* Customer & Product Analysis

## 🔍 Key Insights
* West region generates the highest sales
* Technology category performs best
* Sales vary significantly across months
* Some categories show lower performance and need improvement
